# JukeBot
Discord bot using [Nextcord](https://github.com/nextcord/nextcord/) and [Poetry](https://python-poetry.org/)
___

## Installation
```
git clone https://github.com/Dysta/JukeBot 
cd JukeBot
poetry install
```

## Configuration
Rename `.env.example` into `.env`

## Usage
Run `poetry run python bot.py`
